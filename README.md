
# Live ChatApp

ChatApp is a chatting website built using React and it is devloped to chat with your friends.


## Run Locally

Clone the project

```bash
  git clone https://github.com/the-error4-0-4/ChatApp.git
```

Go to the project directory

```bash
  cd ChatApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
